Career Journey

Why Cloud Security?

My interest in cloud security began during my studies in Computer Network Security. Through learning networking, cybersecurity, and cloud computing concepts, I discovered how important cloud security is in protecting modern digital infrastructures. This field combines security, networking, and cloud technologies, making it a career path that matches my interests and long-term goals.

Career Goal

Cloud Security Engineer

A Cloud Security Engineer is responsible for securing cloud infrastructure, protecting sensitive data, managing access controls, and ensuring cloud services remain safe from cyber threats.

Key Responsibilities

🔐 Cloud Infrastructure Security

Protect cloud resources and services from cyber threats and unauthorized access.

👤 Identity & Access Management

Manage user authentication, permissions, and access controls.

📊 Security Monitoring

Monitor cloud environments and detect suspicious activities.

⚠️ Incident Response

Investigate and respond to security incidents effectively.

Core Skills

AWS Azure Linux Python Networking IAM SIEM Risk Assessment

Why This Career Matters

Cloud Security Engineers play an important role in protecting cloud-based systems, applications, and sensitive information from cyber threats. As organizations increasingly adopt cloud technologies, securing cloud environments has become a critical priority to ensure confidentiality, integrity, and availability of digital assets.
